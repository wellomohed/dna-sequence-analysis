# DNA Sequence Analysis

A beginner Python project that reads a DNA sequence from a FASTA file and performs basic sequence analysis.

## Features

- Reads DNA sequences from FASTA files
- Calculates sequence length
- Calculates GC percentage
- Validates DNA bases
- Identifies invalid characters
## How to Run

Make sure `sequence.fasta` is in the same folder, then run:

```bash
python seq_analysis.py
